%\section{Organização}

A equipe responsável pelo desenvolvimento deste projeto está organizada em 5 grupos:
%(vide organograma do projeto em anexo):

\begin{enumerate}
  \item Grupo de mecânica. Responsável pelo projeto e especificações dos componentes mecânicos do sistema
      robótico. Este grupo é composto por
      \begin{itemize}
        \item 1 coordenador (professor doutor);
        \item 1 engenheiro (aluno de mestrado) e
        \item 1 empresa de consultoria (Rodrigo).
      \end{itemize}

  \item Grupo de eletrônica. Responsável pelo projeto e especificações dos componentes eletrônicos do
      sistema robótico. Este grupo é composto por
      \begin{itemize}
        \item 1 coordenador (professor doutor);
        \item 1 engenheiro (aluno de mestrado) e
        \item 1 aluno (mestrado).
      \end{itemize}

  \item Grupo de potência. Responsável pelo projeto e especificações do sistema de alimentação do sistema
      robótico. Este grupo é composto por
      \begin{itemize}
        \item 1 coordenador (professor doutor) e
        \item 1 engenheiro (aluno de mestrado).
      \end{itemize}

  \item Grupo de software. Responsável pelo projeto e codificação da interface gráfica do sistema robótico.
      Este grupo é composto por
      \begin{itemize}
        \item 1 coordenador (professor doutor);
        \item 1 engenheiro (aluno de mestrado) e
        \item 3 alunos (graduação e mestrado).
      \end{itemize}

  \item Grupo de processamento de sinais. Responsável pelo desenvolvimento dos algoritmos necessários para
      o tratamento dos sinais aquisitados pelos sensores do sistema robótico. Este grupo é composto por
      \begin{itemize}
        \item 2 coordenadores (professores doutores);
        \item 1 engenheiro (pós-doutorado) e
        \item 5 alunos (mestrado, doutorado e pós-doutorado).
      \end{itemize}
\end{enumerate}
