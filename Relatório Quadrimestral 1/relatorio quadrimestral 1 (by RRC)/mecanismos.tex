\section{Mecanismos de acompanhamento}

\subsection{Execução}

Para monitorar a evolução do cronograma de execução do projeto, são utilizados os seguintes mecanismos:

\begin{enumerate}
  \item Reuniões internas dos membros de cada grupo. \\
  Estas reuniões são agendadas à conveniência dos coordenadores dos grupos.
  Em geral, são realizadas semanalmente.

  \item Reuniões dos coordenadores. \\
  Estas reuniões são realizadas sempre às 2a. feiras exceto nos casos em que não haja pauta relevante.
  Nessas reuniões são realizadas deliberações, planejamento de atividades e acompanhamento dos trabalhos
  realizados por cada grupo.

%%%  \item Reuniões com a ESBR. \\
%%%  Estas reuniões são realizadas mensalmente com a participação do coordenador do projeto e representante
da ESBR. Eventualmente, um ou mais coordenadores de grupo são convidados para apresentarem os resultados
com mais detalhes.
%%%  Nessas reuniões são avaliadas as atividades realizadas durante o mês anterior.

  \item Seminários internos. \\
  A cada evento, um grupo diferente faz uma apresentação de seus resultados.
  %Estes são organizados e apresentados pelos integrantes do grupo.
  O propósito deste evento é fornecer uma visão global do projeto para todos os participantes
  e ter a oportunidade de discutir aspectos do projeto que são comuns a mais de um grupo.

  \item Relatórios técnicos. \\
  Estes relatórios documentam de forma detalhada todas as atividades de pesquisa e desenvolvimento
  realizadas pelo grupo. Incluem praticamente tudo o que o grupo realizou desde o início dos trabalhos. Os
  relatórios são atualizados pelo menos uma vez por mês. Os incrementos são realizados mais ou menos de
  forma cronológica. Todo o material pesquisado e coletado deve ser registrado ou referenciado.

  \item Relatórios resumidos. \\
  São relatórios de uma ou duas páginas que relatam de forma resumida as atividades de pesquisa e
  desenvolvimento realizadas pelo grupo no último mês. Estes relatórios são avaliados nas reuniões dos
  coordenadores no final do mês.

  \item Planejamento semestral. \\
  Os grupos realizam um planejamento detalhado das atividades do semestre.
  Neste planejamento, as tarefas são discriminadas mês a mês.
  O propósito é servir de orientação para as atividades do grupo.
  Possíveis refinamentos e alterações são apresentados nos relatórios resumidos.

  \item Repositório de arquivos. \\
  Primeiramente, este repositório local permite assegurar a confidencialidade dos documentos mantidos pela
  equipe. Além disso, o acesso a este repositório permite um acompanhamento praticamente \emph{on line} de
  todas as atividades realizadas.

\end{enumerate}

\subsection{Financeiro}

Para o acompanhamento da movimentação financeira do projeto são utilizados basicamente dois instrumentos:

\begin{itemize}
  \item Relatório financeiro resumido mantido pela coordenação do projeto e
  \item Relatório financeiro completo disponibilizado pela Fundação COOPETEC.
\end{itemize}
